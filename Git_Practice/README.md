## 1. Git Mastery

Ziel-Themen:

- Interacive Rebase: Commits sauber machen(squash, rename, reorder)
- Cherry-Pick: einen Commit aus einem anderen Branch übernehmen
- Bisect: in einem Test-Repo einen fehlerhaften Commit finden
- Eigene Comit-Historie der letzten 2 Wochen reviewn - war sie sauber?
